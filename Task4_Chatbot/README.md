# Task 4: Basic  Chatbot

A simple rule-based chatbot built in Python for my CodeAlpha internship.

## Features
- Responds to "hello", "how are you", and "bye"
- Case-insensitive input handling
- Handles unknown inputs gracefully
- Clean exit on "bye"

## How to Run
```bash
python chatbot.py
```

## Sample Conversation
```
Welcome to CodeAlpha Chatbot!
Type 'bye' to exit.

You: hello
Bot: Hi there! How can I help you?
You: how are you
Bot: I'm doing great, thanks for asking!
You: bye
Bot: Goodbye! Have a nice day!
```

## Technologies Used
- Python functions
- if-elif-else logic
- while loops
- String methods (`.lower()`, `.strip()`)
