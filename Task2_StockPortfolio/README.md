# Task 2: Stock Portfolio Tracker

A simple stock portfolio calculator built in Python for my CodeAlpha internship.

## Features
- 5 predefined stocks with hardcoded prices
- User inputs stock symbol and quantity
- Input validation for stock symbol and quantity
- Calculates total investment value
- Saves results to portfolio.txt
- Multiple stock entries in one session
- Formatted output with separators

## How to Run
```bash
python stock_portfolio.py
```

## Sample Output
```
Enter stock symbol (or 'exit' to quit): AAPL
✅ Stock is available!
Enter quantity: 10

📊 INVESTMENT SUMMARY
Stock:            AAPL
Price per share:  $180
Quantity:         10 shares
Total Investment: $1800
💾 Saved to portfolio.txt!
```

## Technologies Used
- Python dictionaries
- while loops (nested)
- try-except error handling
- File handling (append mode)
- String formatting (f-strings)
