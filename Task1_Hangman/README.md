# CodeAlpha Hangman Game

A simple text-based Hangman game built in Python for my CodeAlpha internship.

## Features
- 5 predefined words
- 6 lives with visual ASCII hangman figure
- Input validation (single letter, alphabetic, no duplicates)
- Play again option

## How to Run
```bash
python hangman.py
```

## Technologies Used
- Python 3.x
- random, while loops, if-else, strings, lists
